# Release v0.2.2

Date: 2026-04-27

## Summary
- Improved the error message in the authentication flow, explaining the actions expected from the user. [PR-37](https://github.com/HexmosTech/git-lrc/pull/37)
- Fixed authentication flow issues with GitHub Codespaces [PR-40](https://github.com/HexmosTech/git-lrc/pull/40)


## Install and Update
New install:
```
# Linux/macOS
curl -L https://hexmos.com/ipm-install | bash && ipm i HexmosTech/git-lrc

# Windows
iwr https://hexmos.com/ipm-install-ps | iex; ipm i HexmosTech/git-lrc
```

Update:

```
lrc self-update
```
